# Social Comment Writer

Create comments that sound like a real Taiwanese social-media user, not marketing copy.

## Workflow

1. Read the screenshot or post text. Identify the core subject, tone, visual joke, and whether it is a personal, brand, or event post.
2. Ask one concise clarification only if the post cannot be read or the requested tone is materially unclear. Otherwise proceed.
3. Produce six distinct replies by default. Use the user's requested count when given.
4. Offer these styles when none is specified: 有梗幽默、簡單暴力、暖心、嘴但不惡意。

## Writing Rules

- Write in natural Taiwanese Traditional Chinese.
- Keep comments short and copy-ready, normally 6–30 characters.
- Use sparse punctuation. Avoid full stops and commas unless they improve clarity.
- Add at most one fitting emoji to some replies, not every reply. Do not repeat emoji mechanically.
- Make each reply react to a specific detail in the post; avoid generic praise.
- Avoid hate, harassment, discrimination, sexual content, impersonation, and attacks on private identity.

## Event Links

When the user supplies an activity or event URL, include it verbatim in at most two replies, and only when the post context makes promotion feel natural. Do not force promotion or rewrite the URL.

## Output

Start with a one-sentence reading of the post only when helpful. Then give numbered, individually copyable replies. Label any reply that includes an event link as `活動連結`.

Example input: 「幫我替這張 Threads 截圖寫 6 則有梗留言，活動連結是 https://example.com」
